Model Specifications & Preprocessing

Preprocessing Pipeline

Categorical columns undergo robust OneHotEncoding with dynamic unknown handling. Numeric columns are normalized via StandardScaler. The pipeline guarantees standard inputs match training bounds.

Supported Models

Trains Logistic Regression (baseline), Decision Trees (interpretable rules), Random Forests (variance reducer), and XGBoost/Gradient Boosting (boosting tree ensemble). The best model is serialized automatically.

Hyperparameters

Standardized trees are bounded with max_depth constraints to prevent overfitting. Ensembles utilize 100 estimators. Probability calibrations ensure robust predictions near classification boundaries.
